MyAuthApp
Configurer l'authentification à deux facteurs

Cliquez pour générer le QR code, puis scannez-le avec votre application TOTP (Google Authenticator, Authy...)

Scanne le QR avec ton appli, puis entre le code généré pour vérifier.

Clé (au cas où) :

Après activation, tu devras fournir le code TOTP à chaque connexion.
